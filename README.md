# 🥤 BajaBlast Breakout 🚀  

A refreshing twist on the classic 3D **Breakout** game, made in Unity! Aim for the high score! 🍹💥  

## 🎮 How to Play  
- 🏓 Use your **paddle** to bounce the ball.   
- 🚀 Fast reflexes required!  

## 🛠️ Built With  
- 🎮 **Unity**  
- 🔥 **C#**
